# Workflow

## Monthly Process

1. Export VoC data from Notion DB as CSV.
2. Update the Excel master workbook.
3. Run Claude Skill to classify, group, summarize, and draft report sections.
4. Review ambiguous items manually.
5. Export the monthly HTML report.
6. Save the monthly snapshot for future comparison.
7. Record corrections and feedback in `feedback-log.md`.

## Review Criteria

- Are similar VoC items grouped by meaning rather than surface keywords?
- Are high-severity or repeated issues visible in the summary?
- Are unresolved and planned items clearly separated?
- Are report labels clear about denominator and population?
- Are any sensitive customer details removed before sharing?
