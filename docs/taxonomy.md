# Taxonomy

This document tracks the working classification rules for VoC review.

## Category Fields

| Field | Purpose |
| --- | --- |
| Main category | Product, operation, education, account, data, or other high-level domain |
| Subcategory | More specific issue type |
| Status | New, unresolved, planned, completed, deferred, or monitoring |
| Severity | Impact level for customer operation or product reliability |
| Owner | Team or role responsible for follow-up |

## Update Rule

When a new VoC does not fit the existing taxonomy, record the proposed category, reason for addition, and reviewer decision before using it in the next monthly report.
